## Install

```sh
brew install fx
```
```sh
snap install fx
```
```sh
scoop install fx
```
```sh
pacman -S fx
```
```sh
pkg install fx
```
```sh
go install github.com/antonmedv/fx@latest
```

```sh
npm install -g fx
```

```sh
curl https://fx.wtf/install.sh | sh
```

## Usage

Start the interactive viewer via:

```sh
fx data.json
```

Or

```sh
curl ... | fx
```

Pretty print:

```sh
curl ... | fx .
```
