
```sh
snap install fx
```

```sh
scoop install fx
```

```sh
pacman -S fx
```

```sh
pkg install fx
```

```sh
go install github.com/antonmedv/fx@latest
```

## Documentation

See full documentation at [fx.wtf](https://fx.wtf).

## Usage

Start the interactive viewer via:

```sh
fx data.json
```

Or:

```sh
curl ... | fx
```

Pretty print:

```sh
curl ... | fx .field
```
